键盘 -、+控制螺旋桨的转速，按住按键持续加或者降低转速
↑、↓键增加或减少升力（调整主螺旋桨攻角，攻角大空气流速快，会产生更大的升力）

w、s键控制机身前后俯仰
a、d键控制机身左右翻转

起飞基本操作：
先按住+键，把转速拉到最大，此时飞机是不会升起来的，还要按↑键产生升力，飞机就会离地，因为在升力的持续作用下
飞机会一直上升，为了达到空中悬停的目的，需要点按↓键去减小升力，如果飞机下降就点按↑键增加升力，如此反复↑、↓键点按以达到动态平衡

w、s键控制机身前后俯仰，会让飞机前后移动，a、d键控制机身左右翻滚，会让飞机左右移动，目前的操作控制飞机移动还是比较有难度的，后续会考虑怎么简化它
